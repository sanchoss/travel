- [ ] Сфотографувати показник одометра (пробіг)
- [ ] Сфотографувати рівень палива на приладовій панелі
- [ ] Сфотографувати номерний знак

- [ ] Перевірити кондиціонер/обігрівач

- [ ] Перевірити ремені безпеки на всіх 4 місцях
- [ ] Налаштувати сидіння і кермо

### 4. Технічне
- [ ] Перевірити фари, поворотники, стоп-сигнали, аварійку
